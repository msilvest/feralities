# feralities
3-d tetris project

# Instructions
1. Install Node.js: https://nodejs.org/en
2. Install three.js and vite
    # install
    npm install

    # individual packages
    three.js <br />
    npm install --save three <br />
    vite <br />
    npm install --save-dev vite
4. Start
    # start
    npm start
